# Student Report System (Python)

This is a simple Python project built as part of my learning journey.

## 📌 Features
- Takes marks as input from user
- Calculates total and average
- Finds highest and lowest marks
- Counts pass and fail students
- Assigns grade based on performance

## 🧠 Concepts Used
- Functions
- Loops
- Conditions
- Lists
- Modular programming

## ▶️ How to Run
1. Open the file in Python
2. Run the program
3. Enter number of students
4. Enter marks
